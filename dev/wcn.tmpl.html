Wander Console Network

Wander is a small, decentralised, self-hosted web console that lets visitors to your website explore interesting websites and pages recommended by a community of independent personal website owners. To learn more about Wander Console, please visit codeberg.org/susam/wander.

This page presents a snapshot of known consoles and recommended web pages from the Wander Console Network. The snapshot was produced by a crawler that explores the network starting from a set of known Wander consoles. As a result, it may not include every console in the network, especially those that are not linked from the starting points. This snapshot was generated on @date. The data shown on this page is also available in a JSON file here: wcn.json.
